# node index.js

# (переименовано, в оригинале server.js т.к. отвечает за серверную часть приложения)

/_/ Проект на котором работает Concurrently, который позволяет нам запускать серверную часть приложения(папка concurrently)
и клиентскую (папка client) на разных портах (3000 для клиентской части и 8080 для серверной)
Необходимо нам все это для проекта FULLSTACK SICK - FITS из уроков от WESBOS, нужно перенести его приложение сюда,
и сделать так чтобы оно работало ! /_/

# Когда вернешься к проекту sick-fits не забудь прочитать это!

1. Думать своей головой, понимая что ты делаешь и зачем
2. Загрузить пакеты последних версий
3. Написать минимальный код или оставить для проверки существующий
4. Плавно и по частям (а лучше создавая с нуля) перенести пакеты по нужным папкам, не забывая соблюдать структуру существующего приложения
